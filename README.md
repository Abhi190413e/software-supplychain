# software-supplychain
This is a python based CLI that checks the details of the installed libraries and packages like version , tampering and the known vulnerabilities.
Steps to use this tool:
       1. clone this repository.
       2.create a virtual environment.
       3.activate the venv.
       4.import the required libraries to run the tool.
       5.run the supplychain.py file



#Report:
  The scan report is now saved in the report.json file.
